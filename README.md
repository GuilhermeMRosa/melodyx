# Melodyx

O **Melodyx** é um aplicativo de streaming de músicas desenvolvido em JavaScript como atividade avaliativa para a disciplina de Engenharia de Software II, no curso de Análise e Desenvolvimento de Sistemas da Fatec SJC.

---

## Objetivo

O projeto tem como objetivo simular um aplicativo de streaming de música, permitindo que usuários possam:

* Ouvir músicas
* Criar e gerenciar playlists
* Interagir com músicas (favoritar)
* Compartilhar playlists
